# ChatApplication
To use this application you need to install the following pod file :-
 1) pod 'Firebase/Firestore'
 2) pod 'Firebase/Database'
 3) pod 'Firebase/Auth'
 4) pod 'Firebase/Storage'
 5) pod 'lottie-ios'
